설정 방법
  1. Supabase 프로젝트에서 Project URL과 anon key를 복사합니다.
  2. "⚙ Supabase 설정 입력" 버튼을 눌러 두 값을 입력합니다. (브라우저 localStorage에만 저장됨)
  3. Authentication → URL Configuration에서 현재 페이지 URL을 Redirect URL로 등록합니다.
  4. Authentication → Providers에서 Google과 Kakao를 활성화하고 각 OAuth 키를 입력합니다.
  5. Google: Google Cloud Console에서 OAuth 2.0 클라이언트 생성.
  6. Kakao: Kakao Developers에서 앱 생성 후 카카오 로그인 활성화.

※ 이 샘플은 클라이언트 사이드만 사용합니다. 비밀 키는 절대 코드에 직접 넣지 마세요.
