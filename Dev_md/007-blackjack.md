## 왜 블랙잭인가

카드게임 후보를 비교했을 때:

| 게임 | 장점 | 단점 |
|---|---|---|
| 솔리테어(Klondike) | 가장 유명 | 드래그앤드롭 + 4개 더미 + 7개 컬럼 복잡 |
| **블랙잭** | **빠른 게임 루프(1-2분)** · 단일 vs 딜러 · **명확한 UI** | — |
| 포커 | 패 평가 흥미로움 | 핸드 평가 로직 + 베팅 라운드 복잡 |
| 워 | 매우 단순 | 사용자 의사결정 없음(재미 ↓) |

블랙잭이 "한 페이지 안에 끝나는 카드게임 샘플"로 가장 균형이 좋다.

## 핵심 규칙 구현

- **카드 값**: A=1 또는 11(버스트 시 자동 1), J·Q·K=10, 나머지는 숫자
- **딜러**: 17 미만이면 무조건 hit, 17 이상이면 stand
- **블랙잭**: 첫 2장이 21이면 **1.5배 배당**
- **일반 승리**: 2배 배당
- **무승부(푸시)**: 베팅금 반환
- **더블다운**: 첫 2장에서만, 베팅 2배 + 카드 1장만 추가 후 자동 stand

## 구현 노트

### 카드 디자인 (SVG 없음)
순수 CSS만으로 4개 슈트(♠♥♦♣) 카드를 그렸다:
- 좌상단: 랭크 + 슈트 작게
- 중앙: 거대한 슈트 (40px, 85% 투명도)
- 우하단: 좌상단을 `transform: rotate(180deg)`로 뒤집어 동일하게

```css
.card .corner-br { transform: rotate(180deg); /* 좌상단 그대로 재사용 */ }
```

### 카드 뒷면
딜러의 두 번째 카드는 시작 시 뒷면으로 표시. `.card.back` 클래스가 그라데이션 배경 + 거대 ♠ 심볼 패턴.

### 베팅 칩
4개 단위(10/50/100/500)를 카지노 칩처럼 원형 + 점선 테두리로 디자인. 색은 카지노 표준(블루/오렌지/블랙/레드).

### 카드 딜링 애니메이션
새 라운드 시작 시 `.cards.dealing .card` 셀렉터에 `@keyframes deal`이 적용 — Y축 -80px + 12도 회전에서 원위치로 350ms 등장.

### A 처리
```js
function valueOfHand(cards) {
  let v = 0, aces = 0;
  for (const c of cards) {
    if (c.rank === 'A') { v += 11; aces++; }
    ...
  }
  while (v > 21 && aces > 0) { v -= 10; aces--; }  // 버스트면 11→1
  return v;
}
```

### 칩 영속화
- `localStorage.blackjack-chips`: 현재 보유 칩 (시작 시 1,000)
- `localStorage.blackjack-best`: 최고로 보유했던 칩 수
- 0원이 되면 confirm으로 1,000원 보너스 옵션 제공 (게임 종료 방지)
