## 모던 테트리스 기능 포함

기본 클론에서 끝내지 않고 1984년 원작 이후 도입된 표준 기능을 함께 구현했다:

- **7가지 테트로미노**: I · O · T · S · Z · J · L 각자 색상 + 회전 모양 자동 계산
- **7-bag 랜덤**: 7개 블록을 한 번씩 다 본 다음에야 새 가방. 운빨에 의한 같은 블록 연속 등장 방지
- **NEXT 3개 미리보기**: 우측 패널에 다음 블록 3개 표시
- **HOLD**: C 또는 Shift로 블록 보관, 한 번 사용하면 다음 블록 락 전까지 못 씀
- **고스트(ghost)**: 현재 블록이 떨어질 위치를 18% 투명도로 미리 표시
- **하드드롭**: 스페이스로 즉시 바닥까지 (떨어진 칸 × 2 보너스 점수)
- **wall-kick**: 회전 시 벽이나 블록에 닿으면 ±1, ±2칸 이동 시도
- **고전 점수 체계**: 1/2/3/4줄 동시 제거 시 100/300/500/800 × 레벨
- **레벨업**: 10줄마다 +1 레벨, 낙하 속도 60ms 단축 (최소 80ms)

## 구현 노트

### Canvas 셀 그리기
한 셀(`drawCell`)을 정사각형 + 상단 4px 하이라이트로 그려서 입체감 부여. 고스트는 같은 함수를 alpha=0.18로 호출.

### 게임 루프
`requestAnimationFrame` 기반. 매 프레임마다 delta 누적해 dropInterval 초과 시 한 칸 낙하. paused 플래그로 일시정지 시 lastTime 초기화 → 재개 시 즉시 낙하하지 않도록.

### 회전과 wall-kick
```js
function rotate(blocks) {
  // 시계방향 90도: out[c][N-1-r] = in[r][c]
}
function tryRotate() {
  const rot = rotate(current.blocks);
  for (const dx of [0, -1, 1, -2, 2]) {  // 0칸 → 좌1 → 우1 → 좌2 → 우2
    if (!collides(currentX + dx, currentY, rot)) { ... return; }
```

### 입력
- 키보드: ← → ↓ 이동, ↑ 회전, 스페이스 하드드롭, P 일시정지, C/Shift 홀드
- D-Pad: 모바일을 위해 6개 버튼 (좌·우·회전·아래·드롭·홀드)
- 키 충돌 회피: 게임 시작 전 스페이스는 start, 게임 중 스페이스는 hardDrop
