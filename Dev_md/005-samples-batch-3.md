## 이번 배치의 패턴

### 1. "한 줄로 표현되는 미션" 우선
각 샘플이 무엇인지 한 줄로 정확히 떠올릴 수 있도록 시각적 정체성을 명확히 잡았다.
- `personal-03` → "Linktree"
- `landing-03` → "페스티벌 카운트다운"
- `game-06` → "PONG"

### 2. 입력 디바이스 다양화 (게임)
- `game-06` PONG: 마우스 + 키보드 + 터치 3종 입력
- `game-07` 두더지 잡기: 클릭/탭만 — 의도적으로 단순 (어린이도 즐기는 게임)

### 3. 실시간 시뮬레이션 패턴 재사용
- `data-06` 코인: 3초마다 ±0.3% 가격 흔들기로 거래소 분위기
- `data-07` IoT: 24시간 환경/전력 차트는 정적이지만 토글은 즉시 반영
- `landing-03` 페스티벌: 1초마다 카운트다운 갱신

### 4. 글래스모피즘 부활
`personal-03`에서 backdrop-filter + 반투명 카드 + 그라데이션 배경 조합으로 2020년대 SNS 인플루언서 페이지 분위기를 그대로 재현.

## 구현 노트

### `personal-03` — Linktree
- CSS columns 안 쓰고 단순 flex column으로 모바일 우선
- 클릭 카운터를 localStorage에 영속화 (사이트 운영자가 추적하는 척)
- `lk-feature` 클래스로 핵심 링크 1개를 다크 카드로 강조

### `landing-03` — 페스티벌
- 네온 핑크(`#ff2e88`) + 시안(`#00e0ff`) + 옐로우(`#ffe04a`) 3색 그라데이션
- 헤드라이너 폰트 크기를 `clamp(70px, 14vw, 180px)`로 화면 가득
- 라인업은 `.row--xl/.row--l/.row--m` 클래스로 폰트 크기 위계 차별화

### `app-06` — 운동 트래커
- SVG `r=44`, `circumference=276`을 dasharray로 고정, dashoffset만 애니메이션
- 시작 버튼이 운동 미완료 항목을 자동 선택해 토글 — 사용성 강화
- 전체 완료 시 버튼이 "🎉 운동 완료!" 상태로 전환

### `app-07` — 캘린더
- 첫 요일(`new Date(y, m, 1).getDay()`) + 이전 달 마지막 날(`new Date(y, m, 0).getDate()`)로 빈 칸 채우기
- 일정 데이터는 `{ '2025-5-24': [{title, time, cat}] }` 구조로 단순화
- 카테고리별 4색(`work/personal/study/health`) 점이 셀 하단에 표시

### `ai-06` — 이미지 편집
- AI 도구 클릭 시 4단계 진행 텍스트가 ms 단위로 교체("인코딩→추론→디코딩→완료")
- CSS `filter` 속성(`brightness`, `contrast`, `saturate`, `hue-rotate`)으로 실시간 조정
- 작업 후 히스토리 사이드바에 자동 추가 (최대 6개 유지)

### `ai-07` — 이력서 첨삭
- before/after 첨삭 예시는 코드 diff 스타일로 색상 분리(주황=BEFORE, 청록=AFTER)
- 5단계 로딩 텍스트(`이력서를 읽는 중→키워드 매칭→...`)로 LLM 추론 흉내
- 직무 선택 셀렉트는 실제 환경에서 다른 평가 프롬프트로 분기 가능

### `data-06` — 코인 대시보드
- 보유 자산 계산: 각 코인의 가짜 평단가(현재가에서 변동률만큼 역산) × 보유 수량
- 코인별 가격 차트 데이터 생성기: 시드 기반 랜덤 워크
- `setInterval`로 3초마다 시세 흔들기 — 인터넷 끊겨도 동작하는 데모

### `data-07` — 스마트홈
- 4개 방 × 3-4개 기기로 가족 단위 스마트홈 시뮬레이션
- 기기 토글 시 `.on/.off` 클래스 즉시 전환 — 시각적 피드백 강조
- 차트는 Chart.js dual Y-axis로 온도/습도를 한 그래프에 표시

### `game-06` — PONG
- AI는 공의 Y좌표를 `lerp(0.045-0.11)`로 추적 — 약한 추적 = 사람이 이길 수 있음
- 패들 충돌 시 (맞은 위치 - 패들 중앙) / 패들 반높이 = -1~+1 → 반사각 결정
- 매 충돌마다 공 속도 ×1.04 → 게임이 점점 어려워짐

### `game-07` — 두더지
- Canvas 없이 DOM + CSS transition만으로 두더지 위아래 애니메이션
- 폭탄 출현 확률 난이도별(10/18/25%) 차등
- 콤보 3회마다 점수 +1 가산 → 빠르게 연타하는 게 핵심
